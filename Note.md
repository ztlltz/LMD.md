# 1.字体
#### 粗体
使用`**内容**`来让**内容**加粗-> **内容**
#### 斜体
使用`_内容_`来让**内容**变歪->_内容_  

# 2.标题
在内容前加`#`，#越多标题级数越低，字体越小，最多六级,会自动换行  
># 一级标题
>`# 一级标题`
>## 二级标题
>`## 二级标题`
>### 三级标题
>`### 三级标题`
>#### 四级标题
>`#### 四级标题`
>##### 五级标题
>`##### 五级标题`
>###### 六级标题
>`###### 六级标题`

# 3.超链接
有两种方法使用超链接  
```
1. [内容](链接)

2. [内容][代号]

   [代号]:链接
```
两种最终的效果是一样的：

[我是一个超链接](https://github.com/ztlltz)

[巧了我也是一个超链接][代号1]

[代号1]:https://github.com/ztlltz

<h1 id=1> 4.图片</h1>

# 5.引用
有的人可能会问，引用是什么：
>我是引用哦  
写法是使用 **>**

```
单行
>one

多行
>one
>
>two
>
>three
```
效果分别为
>one
___
>one
>
>two
>
>three

# 6.列举
列举可以使用两种，一种是 **点号** ，一种是 **序号**
```
点号表示

* 阿梓
* 小可
* 七海

点号里面也可以包含点号
* vtb
   * 阿梓
   * 小可
   * 七海

1. 阿梓
2. 小可
3. 七海
```
效果：
* 阿梓
* 小可
* 七海  
___
* 虚拟主播
   * 阿梓
   * 小可
   * 七海
___

1. 阿梓
2. 小可
3. 七海

# 8.段落
有两种办法可以让文字分段（或者说叫换行）
```
1.在结尾处打两个连续的空格和回车

段落1
段落2

2.打两个回车

段落1

段落2
```
效果如下：  
* 段落1  
  段落2

* 段落1

  段落2
# 9.代码
直接上例子(在此处的 **-** 在实际写时要换成 **`** )
```
---c++
#include<bits/stdc++.h>
using namespace std;
int main(){
      cout<<"Hello world";
      return 0;
}
---
```
**效果：**
```c++
#include<bits/stdc++.h>
using namespace std;
int main(){
      cout<<"Hello world";
      return 0;
}
```
